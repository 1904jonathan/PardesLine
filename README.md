# PardesLine ‚Äì 3D Mesh & Point Cloud Processing üêá

PardesLine is a Python repository demonstrating **essential 3D mesh and point cloud workflows** using [Open3D](http://www.open3d.org/). Perfect for learning and experimenting with 3D data in **computer vision, robotics, VR/AR, and machine learning**.

---

## Modules

* `01_pointcloud_processing.py` ‚Äì Convert meshes to point clouds, sample points (Uniform & Poisson-disk), visualize, and save.
* `02_mesh_processing.py` ‚Äì Load, visualize, color, subdivide, and simplify meshes.
* `functions.py` ‚Äì Helper functions for point cloud and mesh operations.
* `requirements.txt` ‚Äì Required Python packages.

---

## Installation

```bash
git clone https://github.com/1904jonathan/PardesLine.git
cd PardesLine
pip install -r requirements.txt
```

---

## Usage

Run either module to explore the workflows:

```bash
python 01_pointcloud_processing.py
python 02_mesh_processing.py
```

Outputs (meshes and point clouds) are saved in `C:\PardesLineData` by default.
